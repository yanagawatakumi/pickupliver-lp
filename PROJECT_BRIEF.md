# プロジェクト要件サマリー（確定版）

## プロジェクト
- 名称: PICK UP LIVER 特設LP（複数回運用）
- 形式: 各回LP + 回一覧 + 最新回リダイレクト
- 公開URL: `https://pickupliver-lp.pages.dev`

## 目的
- 配信企画の特設ページを回ごとに保存しながら運用する
- 最新回の視聴導線（ColorSing）への送客

## ターゲットと優先デバイス
- 主対象: 配信視聴ユーザー
- 優先: スマホ表示（SP最適化を重視）

## ページ構成
- `/` : 最新回へリダイレクト
- `/events/` : 回一覧
- `/events/vol-<n>/` : 回別LP
- セクション: ヒーロー / MC / GUEST LIVER

## 掲載データ
- データソース: `docs/データベース.xlsx`
- 回別データ: `content/events/vol-<n>/event.json`
- 一覧・最新回: `content/events/index.json`

## デザイン方針
- 方向性: ポップ・派手・イベント感
- 要素: マルキー、ステッカー、粒子エフェクト、クラッカー演出
- 運用方針: 読める範囲で演出優先

## モーション方針（現行）
- 常時粒子: 有効（速度上限つき）
- 入室時クラッカー: 有効（爆大）
- 定期クラッカー: 無効（停止中）
- クリック時クラッカー: 有効
- `prefers-reduced-motion`: 対応済み

## 公開・運用
- ホスティング: Cloudflare Pages（無料）
- ドメイン: `*.pages.dev`（無料サブドメイン）
- 更新方法: `main` へpushで自動デプロイ
- 想定運用: `vol-3`, `vol-4` を同構成で継続追加
